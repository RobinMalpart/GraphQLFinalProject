# Back-End

## Comment lancer le projet

1. Installer les dépendances :
    ```bash
    npm install
    ```

2. Démarrer le serveur :
    ```bash
    npm start
    ```

3. Ouvrir un navigateur et accéder à l'adresse suivante :
    ```
    http://localhost:4000
    ```

## Technologies utilisées
- TypeScript (avec ESLint & Prettier)
- Node.js
- GraphQL (avec Prisma)
